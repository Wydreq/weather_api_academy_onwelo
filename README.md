
# Weather forecast app

Simple app for displaying current or previous weather forecasts.




## Features

- Searching forecast by 3 different cities
- Showing all searched forecasts from local storage
- Responsive app
- Error validation
- Using API


## Tech Stack

- HTML
- CSS
- JavaScript


## Installation

1. Clone repo
2. Open file 'index.html
